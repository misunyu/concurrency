SUMMARY

A concurrency bug in MySQL-5.1+

 

DETAILS

Some details can also be found at: https://bugs.mysql.com/bug.php?id=59464

This bug is due to a data race.

The function row_vers_build_for_semi_consistent_read(), introduced in the fix of Bug #3300, is checking version_trx->conc_state while not holding kernel_mutex.

Version_trx could be null.
